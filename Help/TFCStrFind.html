Command: StrFind

Format: StrFind TheString FindString [StartIndex] [CaseSensitive]

Returns: Returns the zero-based index of the found string or -1 if it wasn't found.

Tries to find FindString somewhere in TheString. If StartIndex is specified, the search begins at that zero-based index. If CaseSensitive is specified as 0, then the search isn't case sensitive. The search is case sensitive by default.

!!Set FoundIndex @@(StrFind "Hello, World" "World")@@
