Template Files

Command: Find

Format: Find ResultVariable SearchString [Flags ...]

Returns: Nothing.

Searches for SearchString using Flags as the search criteria. If SearchString is found, "1" is stored in ResultVariable. If not, "0" is stored in ResultVariable.

Find Flag Description
MatchForward From the current caret position, a forward match is performed.
MatchBackward From the current caret position, a backward match is performed.
MatchFromStart The match is performed from the start of the file.
MatchWord Whole words only.
MatchCase Case-sensitive.
MatchRegExp Regular expression. 
!!Find DidFind "Else" MatchWord MatchCase
