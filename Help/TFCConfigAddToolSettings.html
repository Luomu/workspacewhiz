Template Files

Command: ConfigAddToolSettings

Format: ConfigAddToolSettings ProjectName ConfigName ToolName Settings

Returns: Nothing.

Adds custom compilation settings to all files executed by ToolName. ToolName is most often CL.EXE or LINK.EXE. The settings are added to the configuration called ConfigName existing in the project ProjectName. To select the active project, use the ~ symbol for ProjectName.

!!ConfigAddToolSettings ~ "@@(ProjectName)@@ - Win32 Debug" "cl.exe" "/Yu%"StdAfx.h%""
