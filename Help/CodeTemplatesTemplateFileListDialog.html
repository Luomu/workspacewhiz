Template File List Dialog

The Template File List dialog shows the list of active template groups.

A new template group is inserted into the list by clicking the Add button. A file browser is displayed allowing selection of any *.WWTpl files.

To remove a template group, select the group from the list and click the Remove button.

Template groups may be edited from within Developer Studio. Although a template group file can be opened by hand in Developer Studio, the recommended way of editing a group is by selecting the group in the list and clicking the Edit button.

Pressing OK will return to the Code Templates dialog.
