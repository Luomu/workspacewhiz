Template Files

Template Section: Code

The Code section is required.

The text within the Code section is the actual code to be pasted into the source file. There are special formatting symbols which may be used, but in its simplest form, it just needs to be a block of code.

!!Code
#include <stdio.h>
   
main()
{
    printf("Hello, world!\n");
}
!!End

Note that the text contained within the Code section is transferred directly to the source file. This means all formatting conventions, including tabs, go with that transfer.

Variables may be used at any point:

#define @@(FileTitle:U)@@_@@(FileExt:U)@@

The above code executes the variable command expressions !!FileTitle and !!FileExt. If the currently open file is Test.h, the result outputted to the Developer Studio source file is:

#define TEST_H
