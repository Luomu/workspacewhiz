Template Files

Command: FileNew

Format: FileNew Filename [CreationType]

Returns: Nothing.

Creates a new file called Filename. CreationType specifies more information about where the file should be created.

Creation Type Description
abs (or empty) Assumes Filename is an absolute path.
dsp Assumes Filename is a relative path that will be placed relative to the current project's .dsp file.
rel Assumes Filename is a relative path. 
!!FileNew MyNewFile.h dsp
