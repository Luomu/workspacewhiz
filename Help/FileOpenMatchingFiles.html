Workspace File Open

Matching Files

As letters are typed in the Filename to match: field, files are incrementally matched and displayed in the file list. The search is not case-sensitive and only alpha-numeric characters are counted. Non alpha-numeric characters can be typed, but they are ignored. (For example, if there is a file called !_ABC1_FW!.doc, it is only necessary to type abc1fw.doc.)

The accompanying Prefix: field can be used to prepend a common file prefix.

To match all files of a given extension, insert a period [.] in the Filename to match: field. If only .h files are needed and they all start with the letter A, it is only necessary to type a.h.

Use the slash key [/] or asterisk key [*] at any point to perform a substring (wildcard) search. To match all files that have the word window in them, type /window.

Finally, a line number may be specified. Insert a colon [:] or a comma [,] at the end of the filename, followed by a line number. To go to AFX.H's line 35, type afx.h:35.
