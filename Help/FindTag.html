Workspace Find Tag

Overview

The Find Tag command (and its variations) is a robust graphical extension to Darren Hiebert's excellent Exuberant Ctags software. Every file in the workspace is quickly parsed to find all identifiers (or tags). Any files specified through the Extra Files dialog are also parsed.

Several variations of the Find Tag command are available. One variation shows all functions in a source file. Another finds the tag under the cursor. There are also commands for cycling through similar tags.

The tag information is stored in a file called ProjectName.Tags.WW in the .dsp file's directory.
