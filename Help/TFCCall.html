Template Files

Command: Call

Format: Call TemplateName [ArgName1 ArgValue1 ArgName2 ArgValue2 ...]

Returns: The value specified by the command Return.

Begins processing another code template at the current position. 

[-Hello World]

!!Params
name Joe Bob
!!End


!!Code

Hello World.  Hello @@name@@!
!!End


[Do Stuff]

!!Code

!!Call "-Hello World"
!!Call "-Hello World" name "Joshua Jensen"
!!End

Processing the template Do Stuff yields the following output:

Hello World. Hello Joe Bob!
Hello World. Hello Joshua Jensen!

The minus symbol at the start of the -Hello World means the template won't show in the template selection tree. Private templates are good to use, but !!Call can execute any template.

While executing the called template, a variable called __Parent exists. It is the name of the calling template.
