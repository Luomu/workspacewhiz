Workspace File Open

Navigation

While in the Filename to match: field, the following keys may be used:

Key Command
Up arrow Moves the matched file list's highlight bar up one position.
Down arrow Moves the matched file list's highlight bar down one position.
Left arrow Moves the caret left within the current field.
Right arrow Moves the caret right within the current field.
Page Up Moves the matched file list's highlight bar up one page.
Page Down Moves the matched file list's highlight bar down one page.
Home Moves the caret to the beginning of the current field.
End Moves the caret to the end of the current field.
Ctrl+Home Moves the matched file list's highlight bar to the top of the list.
Ctrl+End Moves the matched file list's highlight bar to the end of the list.
Shift+key Highlights multiple files or characters, depending on the key Shift is being applied to.
Windows Context Menu key Shows the context menu for the edit field or for the file list.

Clicking OK or hitting Enter loads the file into Developer Studio.
