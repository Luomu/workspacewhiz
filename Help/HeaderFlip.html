Header Flip

Overview

Workspace Whiz's Workspace Header Flip opens the current file's "alternate" file (i.e. from .cpp to .h) by searching workspace-wide for the correct, matching file. The header file doesn't even have to be in the same directory as its related source file! If initial searches can't find the alternate file, the directories specified in the global Include and Source paths are searched.

Header flip preferences may be edited on the Header Flip Preferences dialog.
