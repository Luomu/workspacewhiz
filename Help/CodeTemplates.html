Code Templates

Overview

The Workspace Whiz code template facility is a powerful tool for categorizing reusable snippets of code (or text). End-user input may be obtained through built-in support of HTML dialog pages specific to the individual templates. There are also powerful template constructs that are a simpler alternative to the Developer Studio AppWizard.

Template files, herein known as template groups, are given the file extension of .WWTpl. Template groups may exist anywhere on the hard drive, but Workspace Whiz automatically finds templates in the Workspace Whiz/Templates directory. Template groups existing elsewhere on the hard drive are added through the Template File List dialog. The Code Templates dialog is not available until one or more template files are added to the template file list.

The Code Templates dialog graphically shows all the template groups and their templates in an easy to navigate tree form. Template groups are top-level items in the template tree. Subitems under each template group are governed by the individual template's full name. Further information is found here.

The Code Templates dialog may be bypassed by using the WWTemplateComplete command, a facility where a single word is entered into the text editor, the word is matched against a template code "key", and the template containing that key is run.
