Template Files

Header

Template files (otherwise known as template groups) can hold multiple code templates. It is usually wise to group related code templates in the same file. Template files are known by their .WWTpl extension.

The first line of a Workspace Whiz Template Group File is the category of all the templates contained in the file. This category is shown at the root level of the template selection tree. All templates in the file are grouped under this category.

My First Template File!

Each template exists in a block opened with a left square bracket, followed by the name of the template, and followed by a right square bracket:

[My First Template]

By default, the template selection tree is not expanded. To force a template to fully expand the tree, insert a + (plus) symbol at the start of the template name.

[+My First Template]

To create subtrees within the template selection tree, separate each new branch with the | (pipe) symbol.

[Branch 1|Another Branch|My Cool Template]

To create a private template that won't show in the template selection tree (and whose use will be described below), insert a - (minus) symbol at the start of the template name.

[-My Private Template]
