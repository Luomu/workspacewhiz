Toggling Search Attributes

While typing a tag, special symbols may be entered. The symbols, which may be entered anytime, are evaluated left to right. So if the leftmost symbol turns on case sensitivity and a later one turns it off, case sensitivity will be off for the entire search.

Symbol
 Description
+
 Toggles case sensitivity on. Normally, this symbol doesn't need to be used, as typing a capital letter will turn case sensitivity on automatically. The + symbol may be useful, however, when searching for an all lowercase tag.
-
 Toggles case sensitivity off.
\
 Performs a search for tags of the same length only. Therefore, if cfile is the tag being searched for, CFileFind is not found.
=
 The equivalent of combining the + and \ symbols. Only tags matching the search tag exactly are found.
' (apostrophe)

Without any further specification, the apostrophe cancels all tag types being shown. The letters following the apostrophe identify tag types to be turned on. To continue typing a tag after specifying an apostrophe, insert a second apostrophe.
