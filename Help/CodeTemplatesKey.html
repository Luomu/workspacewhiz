Code Templates

Template Keys

The Code Templates dialog may be bypassed by using the WWTemplateComplete command. Templates are inserted through a code "key" system. This key, a single word, is entered into the text editor, and the WWTemplateComplete command is executed. Assuming a template with the proper key, that template will begin running.

Consider the following template (a detailed description of the template format may be found here):

[If Statement]
!!Key
if

!!Code
if (@@(Caret)@@)
{
}
!!End

In the above example, typing the word if into a text editor and executing the WWTemplateComplete command will replace the word if with the body of code in the !!Code section. The caret is positioned at the @@(Caret)@@ location.
