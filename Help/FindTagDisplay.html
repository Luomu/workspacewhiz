Workspace Find Tag

Controlling Tag Display

File Scope

The Scope section of the Find Tag dialog controls which tags, based on location, are considered for the search. If a file is open, the Current File scope shows tags only in the currently open file. The Current Project scope shows tags existing in files in the active project. The Entire Workspace scope shows every tag from every file in the workspace, including those files selected through the Extra Files dialog.

Displaying Certain Tag Types

The Type: field allows selection of a single tag type for display. If the user wanted to just see the functions in the workspace, then Functions should be selected as the tag type. To return to viewing all types, select [All] from the Type: field.

The Type: field only allows a single type to be shown. The Show button allows each type to be toggled on or off individually.
