Template Files

Command: ProjectNew

Format: ProjectNew ProjectName PathToProject ProjectType

Returns: Nothing.

Creates a new project. ProjectName is the name of the new project.

ProjectPath specifies the directory to place this project in. The project (.dsp) and workspace (.dsw) both called project name will be created at the directory path specified via ProjectPath.

ProjectType can be one of the following:

Project Type Description
Application A Win32 executable application
DLL A Win32 Dynamic-link Library
Console Application A Win32 console application
Static Library A Win32 static link library
Utility A utility project
Makefile A makefile project 
!!ProjectNew MyProject c:\MyProjects\MyProject Application
