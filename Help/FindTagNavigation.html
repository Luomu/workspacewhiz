Navigation

While in the Tag to match: field, the following keys may be used:

Key Command
Up arrow Moves the tag list's highlight bar up one position.
Down arrow Moves the tag list's highlight bar down one position.
Left arrow Moves the edit caret left.
Right arrow Moves the edit caret right.
Page Up Moves the tag list's highlight bar up one page.
Page Down Moves the tag list's highlight bar down one page.
Home Moves the caret to the beginning of the edit field.
End Moves the caret to the end of the edit field.
Ctrl+Home Moves the tag list's highlight bar to the top of the list.
Ctrl+End Moves the tag list's highlight bar to the end of the list.
Shift+key Highlights multiple tags or characters, depending on the key Shift is being applied to.
Windows Context Menu key Shows the context menu for the edit field or for the tag list.

Clicking OK or hitting Enter loads the file containing the tag into Developer Studio. Holding down the Shift key while pressing OK or Enter copies the currently selected tag into the source file.
