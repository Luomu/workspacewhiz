Introduction

Workspace Whiz is a powerful Visual C++ 6.0 and Visual Studio 2002/2003/2005 add-in designed to improve developers' productivity by allowing quicker source navigation and coding. Workspace Whiz adds a number of commands that may be used on a Visual Studio toolbar or assignable to a key on the keyboard. Each command is individually tweaked for the most efficient keyboard and mouse use.

A short overview of the major Workspace Whiz features follows:

Workspace File Open:

Workspace Whiz's Workspace File Open command incrementally matches any file in the workspace/solution and any additional external workspaces/projects/files added through the Extra Files dialog. To make searching as efficient as possible, it is unnecessary to type file name symbols, such as underscores. The Workspace File Open command is an efficient method of finding and opening any desired file, without regard to location.

Global File Open:

Built using the same, efficient environment as the Workspace File Open command, the Global File Open command incrementally matches any file in the global Include and Source directories.

Workspace Header Flip:

The Workspace Header Flip opens the current file's "alternate" file (i.e. from .cpp to .h) by searching workspace-wide for the correct, matching file. The header file doesn't even have to be in the same directory as its related source file!

Find Tag:

The Find Tag dialog is a robust graphical extension to Darren Hiebert's excellent Exuberant Ctags software. It incrementally matches any identifier in the workspace or any external projects or workspaces specified.

Several variations of the Find Tag command are available. One variation shows all functions in a source file. Another finds the tag under the cursor. Yet another toggles between the declaration and definition of a tag. There are also commands for cycling through similar tags.

File History:

For every Workspace Whiz command executed, the current caret location and file is added to a history. Use of the history is convenient, for instance, when opening a file to a particular tag and restoring the environment to the original source file in a single keystroke.

Autocomplete Tags:

The Tag Complete command is a natural extension of Workspace Whiz's tag support. Partially typed identifiers are automatically completed by the Tag Complete command. Like most Workspace Whiz functions, the Tag Complete command can take advantage of any external projects or workspaces specified.

Code Templates:

Workspace Whiz contains powerful code template functionality for creating reusable snippets of code, categorizing them, and providing easy graphical access to the templates. User input may be obtained through HTML dialog pages which exist with the individual templates. The HTML dialog pages may take advantage of JavaScript to provide an intuitive end-user experience.

There are also powerful template constructs which may be used as a simpler alternative to the Developer Studio AppWizard.
