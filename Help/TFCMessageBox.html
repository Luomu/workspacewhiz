Command: MessageBox

Format: MessageBox Message [Type] [ReturnCodeVariable]

Returns: One of the Return Types specified below..

Displays a message box containing the text in Message.

Type may be one of the following:

Type Description
AbortRetryIgnore Shows a message box with Abort, Retry, and Ignore buttons.
Ok Shows a message box with an OK button. (Default if no Type specified.)
OkCancel Shows a message box with an OK and Cancel button.
RetryCancel Shows a message box with Retry and Cancel buttons.
YesNo Shows a message box with Yes and No buttons.
YesNoCancel Shows a message box with Yes, No, and Cancel buttons.

When the user has selected a button from the message box, ReturnCodeVariable and the return value are filled in with one of the following values:

Return Type Description
Abort The Abort button was pressed.
Cancel The Cancel button was pressed.
Ignore The Ignore button was pressed.
No The No button was pressed.
OK The OK button was pressed.
Retry The Retry button was pressed.
Yes The Yes button was pressed. 
!!MessageBox "This is my message" OK OutVariable
!!MessageBox "You pressed @@OutVariable@@."
