Template Files

Command: ConfigAddFileSettings

Format: ConfigAddFileSettings ProjectName ConfigName Filename Settings

Returns: Nothing.

Adds custom compilation settings to Filename. The settings are added to the configuration called ConfigName existing in the project ProjectName. To select the active project, use the ~ symbol for ProjectName.

!!ConfigAddFileSettings ~ "@@(ProjectName)@@ - Win32 Debug" "StdAfx.cpp" "/Yc%"StdAfx.h%""
